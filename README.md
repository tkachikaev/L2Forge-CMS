# L2Forge CMS 0.7.0

**L2Forge CMS** — бесплатная open-source CMS для серверов Lineage II.

Это ранний рабочий каркас для последовательной разработки, а не готовый публичный релиз.

## Уже есть

- Laravel 13 и PHP 8.3+.
- Отдельная база CMS и отдельное подключение `game` к базе L2J Mobius.
- Система сменных тем `themes/<theme>`.
- Безопасный демонстрационный адаптер `mock` и начальный адаптер `mobius`.
- Главная страница, новости, статус, рейтинг, страницы входа и регистрации.
- SQLite для простого локального запуска.
- Отдельная защищённая административная панель.
- Управление установленными темами.
- Раздел основных настроек сайта: название, описание, логотип, favicon, часовой пояс, email администрации и текст подвала.
- Управление новостями: список, создание, редактирование, удаление из общего списка, черновики и отложенная публикация.
- Картинки-превью, изображения внутри новости, встроенный визуальный редактор и предпросмотр до сохранения.
- Серверная очистка HTML и строгая проверка загружаемых изображений.
- Статический предпросмотр в `preview/index.html`.

## Первый запуск в Windows

Откройте PowerShell в папке проекта:

```powershell
Set-ExecutionPolicy Bypass -Scope Process
.\setup.ps1
```

После успешной установки создайте первого администратора:

```powershell
php artisan l2forge:admin-create
```

Запустите локальный сервер:

```powershell
.\serve.ps1
```

Откройте:

```text
Сайт:    http://127.0.0.1:8000
Админка: http://127.0.0.1:8000/admin
```

`setup.ps1` выполняется при первой установке. Для обычного запуска используется только `serve.ps1`.

## Диагностика

```powershell
.\doctor.ps1
```

Скрипт проверяет PHP, Composer, расширения PHP, `.env`, зависимости, SQLite и служебные каталоги Laravel.

## Обновление проекта

После получения новой версии файлов:

```powershell
.\update.ps1
```

Скрипт сохраняет существующий `.env` и `APP_KEY`, устанавливает зависимости, выполняет миграции и тесты.


## Настройки сайта

Основные настройки доступны по адресу:

```text
http://127.0.0.1:8000/admin/settings
```

В версии 0.7.0 работает вкладка **«Основные»**. Вкладки **«Игровой сервер»** и **«Логин сервер»** подготовлены для следующих релизов.

Логотип и favicon сохраняются в `public/uploads/settings`. Значение текста подвала по умолчанию:

```text
© 2026 L2Forge-CMS
```

## Новости

Управление новостями:

```text
http://127.0.0.1:8000/admin/news
```

Черновики и будущие публикации не показываются на публичном сайте. Картинки сохраняются в `public/uploads/news`, а пути к ним — в базе CMS.

Очистка старых обложек и изображений, которые больше не используются новостями:

```powershell
php artisan l2forge:news-media-clean --dry-run
php artisan l2forge:news-media-clean
```

## Темы

Установленные темы находятся в `themes/<slug>`, а их публичные ресурсы — в `public/themes/<slug>`.

Управление темами:

```text
http://127.0.0.1:8000/admin/themes
```

Загрузка ZIP пока намеренно отключена до реализации безопасной проверки архивов.

## Важно

`GAME_ADAPTER=mock` установлен намеренно. Регистрация и вход пока не записывают ничего в игровую БД. Сначала нужно зафиксировать конкретную версию Mobius, структуру таблиц и алгоритм хранения пароля.

Фоновое изображение стандартной темы используется как макет. Перед публичным релизом его нужно заменить на собственный или лицензированный арт.
