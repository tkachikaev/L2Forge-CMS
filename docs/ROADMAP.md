# Дорожная карта

## Текущее состояние ядра

KaevCMS 0.31.11 содержит стабилизированное ядро модулей, промокоды, веб-инвентарь, нейтральную очередь наград, локализованный каталог предметов, единый GameServer-драйвер L2J Mobius, фундамент аватаров персонажей и выбор аватара личного кабинета. До публичного релиза приоритетом являются аудит, документация, установка и исправление найденных пользователями проблем. Интеграция пожертвований отложена до первых отзывов после публикации.

- [x] Laravel-ядро, установка и обновление в Windows
- [x] отдельная административная панель, администраторы и TOTP-2FA
- [x] динамические языковые пакеты и переводимый контент без полей `*_ru` / `*_en`
- [x] новости, страницы, SEO, изображения и безопасный визуальный редактор
- [x] независимые темы публичного сайта и шаблоны личного кабинета
- [x] пользователи сайта, регистрация, подтверждение email и восстановление пароля
- [x] SMTP, редактируемые почтовые шаблоны и отправка безопасных HTML-писем
- [x] синхронный, асинхронный и сохраняемый в базе режим автоматической почты с проверкой поддержки
- [x] журнал действий, настройки безопасности и системная диагностика
- [x] отдельные подключения LoginServer и GameServer с шифрованием паролей
- [x] драйверы L2J Mobius LoginServer Legacy C1/C4, LoginServer Interlude+ и единый GameServer-драйвер legacy/modern
- [x] мониторинг совместимости БД, сетевых служб и онлайна игроков
- [x] создание игровых аккаунтов, смена пароля и каталог персонажей в личном кабинете
- [x] автоматический каталог и модальный выбор аватара личного кабинета из файлов владельца сайта
- [x] упрощённый кабинет с отдельными разделами персонажей и игровых аккаунтов без смешивания аватаров
- [x] публичная игровая статистика с кэшем и защитой внешней БД
- [x] PHPUnit, Pint, Larastan/PHPStan и отдельные браузерные сценарии Playwright
- [x] единая структура развёртывания `deployment/windows`, `deployment/hosting`, `deployment/vds`
- [x] усиленный Web Installer для чистой установки на PHP/MySQL-хостинг без shell-команд, с безопасным retry, проверкой прав MySQL и split-пакетом для ограниченного shared-хостинга
