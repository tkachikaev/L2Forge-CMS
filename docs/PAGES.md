# Страницы сайта

Раздел **«Контент → Страницы»** предназначен для правил, контактов, политики конфиденциальности, описаний и других информационных материалов.

Адрес панели:

```text
/admin/pages
```

## Возможности

Для каждой страницы доступны:

- отдельный заголовок, адрес и содержимое для каждого включённого языка;
- черновик или опубликованное состояние;
- отображение в главной навигации и подвале;
- порядок сортировки;
- SEO-заголовок и SEO-описание;
- безопасный визуальный HTML-редактор;
- изображения внутри содержимого;
- предпросмотр без сохранения.

Страница языка по умолчанию обязательна. Другие переводы можно оставить полностью пустыми — на публичном сайте будет использован резервный язык.

## Адреса

Обычный адрес страницы:

```text
/pages/server-rules
```

Локализованные варианты:

```text
/ru/pages/pravila-servera
/en/pages/server-rules
/de/pages/serverregeln
```

Адрес задаётся отдельно для каждого языка. Допустимы строчные латинские буквы, цифры и дефисы. Если поле пустое, CMS создаёт адрес из заголовка.

После изменения адреса старые ссылки перестают работать.

## Языки

Редактор не привязан к русскому и английскому. Вкладки строятся по текущему списку включённых языков из раздела **«Настройки → Языки»**.

После установки и включения нового языкового пакета вкладка автоматически появляется у всех страниц. Миграция базы не требуется.

Переводы хранятся в таблице `page_translations` с кодом локали длиной до 10 символов.

## Навигация

Опции **«Показывать в шапке»** и **«Показывать в подвале»** работают только для опубликованных страниц.

Пункты сортируются по полю **«Порядок сортировки»**: меньшее значение выводится раньше.

Тема может использовать переданные коллекции:

```php
$headerPages
$footerPages
```

Для формирования правильной локализованной ссылки используйте:

```php
page_url($page)
```

## Безопасность

HTML очищается на сервере. Разрешены безопасные заголовки, абзацы, списки, ссылки, цитаты, изображения и ограниченное форматирование.

Удаляются скрипты, формы, iframe, SVG, обработчики событий, внешние изображения и опасные URL-схемы.

Загруженные изображения сохраняются в:

```text
public/uploads/pages/content
```

## Очистка неиспользуемых изображений

Изображение, загруженное в редактор, может остаться без ссылки, если администратор закрыл форму без сохранения. Старые неиспользуемые файлы удаляются командами:

```powershell
php artisan l2forge:page-media-clean --dry-run
php artisan l2forge:page-media-clean
```

По умолчанию сохраняются файлы моложе 24 часов. Параметр `--hours` позволяет изменить этот срок.
