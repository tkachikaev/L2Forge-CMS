# Языки

L2Forge CMS включает два готовых языка:

```text
ru — Русский
en — English
```

Настройки находятся по адресу:

```text
/admin/settings/languages
```

## Доступные параметры

Администратор может:

- включать и отключать установленные языки;
- выбирать язык сайта по умолчанию;
- выбирать резервный язык;
- видеть направление письма и полноту системного перевода.

Нельзя отключить все языки. Язык по умолчанию и резервный язык должны входить в список включённых.

## Переключение языка

Публичный сайт использует адреса с кодом языка:

```text
/ru
/en
/ru/news
/en/news
/ru/login
/en/login
```

Старые адреса без префикса сохранены для совместимости. На них CMS использует язык из сессии, настройку пользователя или язык сайта по умолчанию.

Язык администратора и пользователя сохраняется в их учётной записи. Для гостя выбор хранится в сессии.

## Переводимый контент

Для каждого включённого языка можно отдельно настроить:

- название, описание и текст подвала сайта;
- название игрового сервера;
- заголовок, краткое описание и текст новости;
- тему и содержание системных писем.

Дата публикации, обложка, рейты, хроники, режим и технические настройки остаются общими.

Существующий контент при обновлении переносится в русский перевод. Во время работы CMS проверяет только включённые локали: запрошенную, резервную и основную. Отключённый язык не используется как скрытый fallback; если доступного перевода страницы или новости нет, локализованный адрес не показывает текст другой отключённой локали.

## Добавление нового языка

Дополнительные языки устанавливаются владельцем сервера вручную. Веб-загрузка архивов намеренно отсутствует.

Минимальный пакет, например для немецкого языка:

```text
lang/de/language.php
lang/de.json
```

После копирования файлов язык автоматически появится в разделе «Сайт → Языки». Затем его можно включить и заполнить переводимый контент.

Устанавливайте только проверенные пакеты: файл `language.php` является PHP-кодом и выполняется приложением.

Формат пакета описан в [документации разработчика](development/LOCALIZATION.md).

## Удаление языка

Перед удалением файлов пакета сначала отключите язык в админке. Переводы контента в базе не удаляются автоматически, поэтому пакет можно вернуть без потери введённых текстов.

Встроенные русские и английские файлы входят в релиз и проверяются `doctor.ps1`.
